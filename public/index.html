"Shall We Play a Game?" is a prototype in generative interactive fiction using large language and image diffusion models. All the content the player interacts with is generated in real time, creating a unique playthrough on each session. This is an early development build and may have bugs, errors, inconsistencies and / or weird behavior. The interface and gameplay was inspired by the interactive fiction games of the 1980s, which I grew up playing on amber CRTs :)
